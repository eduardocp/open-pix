## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do **not** report it via public GitHub issues.

Instead, please send an email to the maintainer at [eduardo_carisio@hotmail.com](mailto:eduardo_carisio@hotmail.com) (or adjust to your preferred private contact method).

We will analyze the issue and respond as soon as possible.
